# Add Warp Config

## Step 1 مرحله

You need to install the MoneyFly application from [our site](https://dy.moneyfly.top) or [GitHub](https://github.com/moneyfly004/cboard/releases).

شما باید ابتدا نرم‌افزار MoneyFly را از [سایت ما](https://dy.moneyfly.top) یا [گیت‌هاب](https://github.com/moneyfly004/cboard/releases) دانلود کنید.

## Step 2-1: Easy Way

Easy Way but requires another VPN for first-time use
[Click to Import Free Warp Config](https://dy.moneyfly.top/warp)

راه ساده، اما نیاز به یک فیلترشکن دیگر برای اولین بار دارد.  [کلیک برای اتصال به کانفیگ رایگان](https://dy.moneyfly.top/warp)
