# MoneyFly

MoneyFly is a cross-platform proxy and VPN client with account login, subscription sync, multi-device support, node connection, per-app proxy, routing rules, traffic statistics, and automatic updates.

See [README.md](./README.md) for the current feature overview, downloads, and build information.
